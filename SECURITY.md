# Security Policy

## Supported Versions

This project is pre-1.0. Security fixes are applied to the latest `main` branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities through GitHub private vulnerability reporting (Security Advisories) with:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested mitigation

You should receive an acknowledgment within 72 hours. After triage, we will coordinate remediation and responsible disclosure timing.
